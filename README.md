# Tarkhineh Restaurant

**Tarkhineh Restaurant** is a professional online platform for ordering food, designed to provide users with a fast, simple, and high-quality experience. This website focuses on delivering restaurant services online, allowing users to order food from four active branches in **Vanak**, **Ekbatan**, **Aqdasiyeh**, and **Chalus**. Tarkhineh aims to cover all customer needs, from placing orders to tracking the status of food delivery.

## امکانات اصلی سایت

### معرفی کامل سایت ترخینه

سایت ترخینه یک پلتفرم آنلاین حرفه‌ای برای سفارش غذا است که به کاربران امکان می‌دهد تجربه‌ای سریع، آسان و باکیفیت داشته باشند. این وب‌سایت با تمرکز بر ارائه خدمات آنلاین رستورانی طراحی شده و کاربران می‌توانند از چهار شعبه فعال در ونک، اکباتان، اقدسیه و چالوس غذای خود را سفارش دهند. سایت ترخینه تلاش می‌کند تا تمامی نیازهای مشتریان را از ثبت سفارش گرفته تا پیگیری وضعیت غذا پوشش دهد.

1. **ثبت‌نام و احراز هویت**

   - امکان ثبت‌نام کاربران جدید.
   - احراز هویت برای امنیت بیشتر و مدیریت اطلاعات حساب.

2. **سفارش آنلاین غذا**

   - مشاهده منو با توضیحات و تصاویر باکیفیت.
   - امکان ثبت سفارش به‌راحتی و پرداخت آنلاین.

3. **پیگیری وضعیت سفارش**

   - داشبوردی برای پیگیری مراحل آماده‌سازی و ارسال سفارش.
   - به‌روزرسانی لحظه‌ای از وضعیت سفارش کاربران.

4. **مدیریت پروفایل کاربر**

   - امکان ویرایش اطلاعات شخصی.
   - بخش علاقه‌مندی‌ها برای افزودن یا حذف غذاهای موردعلاقه.

5. **مدیریت آدرس‌ها**
   - کاربران می‌توانند آدرس‌های خود را مدیریت کنند.
   - قابلیت انتخاب بین دریافت حضوری یا ارسال به آدرس ثبت‌شده.

## Responsive Design

Tarkhineh Restaurant is fully responsive, ensuring an optimized user experience across various devices, including mobile, tablet, and desktop.

---

## Technologies Used

For the development of this website, the following advanced tools and technologies were utilized:

- **React.js**: Designing modern and interactive user interfaces.
- **React Hook Form**: Managing forms and efficient validation.
- **TanStack/React Query**: Handling server-side data requests, caching, and synchronization.
- **Supabase**: Database management, authentication, and API integration.
- **Tailwind CSS**: Fast and modern design with responsive utilities.
- **React Router DOM**: Routing and navigation management.
- **Moment Jalaali**: Managing Persian dates tailored to local needs.
- **Leaflet**: Displaying geographical locations of branches interactively on maps.
- **Redux Toolkit**: Efficient and centralized state management.
